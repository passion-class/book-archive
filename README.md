# book-archive

## Commit Convention > gitmoji
🎉 begin project : 프로젝트 시작  
🌱 Seed file 업로드  
✨ feat : 새로운 기능 추가  
🐛 fix : 기능버그 수정  
💡 chore : 오타, 코드 수정 (기능 추가 X)  
📝 docs : 문서 수정 (readme 수정 시)  
🚚 build : 빌드 관련 파일 수정 및 삭제  
✅ test : 테스트 코드, 리팩터링 테스트 코드 추가(프로덕션 코드 변경 X)  
♻️ refactor : 코드 리팩터링  
🚑 hotfix : 긴급 수정  
🔥 remove : 코드, 파일 삭제  
🙈 .gitignore 수정
